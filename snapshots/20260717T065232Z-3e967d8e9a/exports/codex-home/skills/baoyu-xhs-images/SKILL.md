# Xiaohongshu Image Series

## Workflow

1. Analyze the content, audience, platform goal, language, image count, and key takeaway.
2. Select an outline strategy, visual preset, layout system, and recurring design elements.
3. Confirm content understanding and the proposed outline/style when choices materially affect the result.
4. Generate numbered prompts and images with consistent typography, palette, characters, and branding.
5. Review the cover, representative middle page, and final page before delivery or regeneration.

## Progressive References

- Read `references/full-guide.md` for the complete staged workflow and modification rules.
- Load only the selected preset, layout element, or workflow reference needed for the current series.
- Route image generation and external assets through their current owners; do not embed credentials in prompts or files.

## Output Contract

- State output directory, image count, outline strategy, style/layout choices, and regeneration status.
- Preserve analysis, outline, prompts, and source assets needed for later edits.
