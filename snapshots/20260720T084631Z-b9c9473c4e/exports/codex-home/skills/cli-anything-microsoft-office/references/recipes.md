# Recipes

## Word Report Pass

1. Inspect paragraphs, tables, sections, and pages.
2. Replace approved phrases.
3. Add headings and tables before index-sensitive paragraph formatting.
4. Apply page setup, headers, and footers.
5. Reinspect, export PDF, and verify pagination.

## Excel Data Workbook

1. Inspect worksheet names and used ranges.
2. Add or select a target sheet.
3. Write values, then formulas, then formatting.
4. Sort/filter only after the complete table exists.
5. Add charts last so their source range reflects final data.
6. Reopen and verify calculated sample values.

## PowerPoint Update

1. Inspect slide and shape names.
2. Add or delete slides first.
3. Add named text, image, table, and shape objects.
4. Update positions and formatting by shape name.
5. Replace text in a bounded slide when possible.
6. Inspect shapes, export PDF, and review layout.

## Multi-App Package

Create or edit the data workbook first, then the report, then the presentation. Preserve immutable source files and use a shared output directory. Validate every artifact separately before reporting the package complete.
