# General Trusted

Reusable GUI patterns verified across distinct apps or UI surfaces.

Trusted general patterns may be tried before app-specific exploration in unfamiliar apps, but must still be verified after each action.

Each entry should include:

- Pattern: the reusable GUI action or selection strategy.
- Successful apps: distinct apps or UI surfaces where it worked.
- Preconditions: required UI state, file type, or control behavior.
- Verification: durable proof required after use.
- Counterexamples: known scenes where it should not be used.

Do not promote a pattern here from a single app success.
